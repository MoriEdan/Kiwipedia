ZeroWikipedia template

This site is a template for ZeroWikipedia, a clone of Wikipedia in ZeroNet.
To create a new hub from this template, visit ZeroWikipedia and choose Don't see you language in list? in sidebar.

If this is your site, and you've just created a new hub, return to ZeroWikipedia via the following link: init-hub.
